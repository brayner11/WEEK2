Variables and Operators

In JavaScript we learned how variables act as containers for storing information, allowing you to assign names to values such as numbers or text. For example, "var age = 25;" creates a variable called 'age' that contains the value 25. Operators, on the other hand, are symbols that perform operations on variables or values. Examples include arithmetic operators such as '+', used for addition, and comparison operators such as '===', which check whether values are equal. These fundamental concepts allow you to efficiently manage and manipulate data in your JavaScript programs.

If Else

if and else are conditional statements used for decision-making in your code.

Switch

In JavaScript, the switch statement provides a way to perform different actions based on different conditions. It's a concise way to handle multiple cases.
